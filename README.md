Performance Analysis of Top Strikers EPL 2025 26
Project Overview
This project provides a specialized analytical tool for comparing English Premier League attackers including Center Forwards and Wingers for the 2025 2026 season. The core of the project is a dynamic Power BI dashboard designed to support scouting departments and performance analysts in making data backed decisions.

Key Features
Dynamic Head to Head Comparison
Users can instantly compare any two players to visualize performance gaps and playing styles.

Advanced Performance Metrics
The analysis prioritizes high value stats such as Non Penalty Goals, Expected Goals xG, and standardized Per 90 metrics for fair evaluation.

Data Driven Scouting
Financial data is integrated with technical performance to provide a holistic view of a player's market value versus their on field contribution.

Data Sources
FBref
Utilized for granular technical statistics and advanced performance data.

Transfermarkt
Used for accurate market valuations, historical transfer fees, and real time squad updates.

Technical Skills Demonstrated
Web Scraping
Automated extraction of multi source data using Python to ensure a robust and up to date dataset.

Data Cleaning and Modeling
Applied advanced Power Query transformations and DAX measures to handle data mapping and complex calculations.

Data Visualization
Created an interactive Power BI interface focused on user experience and clear storytelling through data.
